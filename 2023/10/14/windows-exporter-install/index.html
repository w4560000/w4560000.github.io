    1. 前言

    前言

    紀錄一下 Windows Exporter 安裝流程

    環境

    Windows Server 2019

    下載 Windows Exporter

    下載 windows_exporter-0.24.0-amd64.exe
    Github - windows_exporter releases

    • 將 windows_exporter-0.24.0-amd64.exe 移到指定目錄，在此以 C 槽為例
    • 在該目錄新增 windows_exporter.yml
      collectors:
enabled: "[defaults],iis"
collector:
service:
  services-where: "Name='windows_exporter'"
log:
level: warn

    因 iis 不在預設的收集器中，需額外指定來開啟，若不需收集 iis 資訊則可無視

    將 Windows Exporter 註冊為服務

    以管理員權限開啟 CMD

    # 註冊服務
sc create windows_exporter binpath= "\"C:\windows_exporter-0.24.0-amd64.exe\" --config.file=\"C:\windows_exporter.yml\" " type= own start= auto displayname= windows_exporter

# 啟動服務
sc start windows_exporter

    啟動成功後，開啟 http://localhost:9182/metrics 查看是否正常

    備註:

    # 暫停服務
sc start windows_exporter

# 刪除服務
sc start windows_exporter
