    Windows環境 以hexo + github page 搭建部落格

    Tool > hexo hexo
    發布時間 : 2022-02-01 19:08
    字數:256 觀看數 :
    評論:
    1. 開發環境
    2. 安裝環境
    3. 建立部落格
    4. 修改 部落格樣式 (以 3-hexo 為例)
    5. 上傳至 GitHub Page
    6. 擴充插件

    開發環境

    • Windows 10 (home 20H2)

    安裝環境

    1. 擁有 Github 帳戶 https://github.com/
    2. 安裝 Node.js (需使用到 npm ) https://nodejs.org/zh-tw/download/
    3. 安裝 Git https://git-scm.com/downloads
    4. 安裝 hexo
      npm install hexo-cli -g

    建立部落格

    ``` PowerShell
hexo init test
cd test
hexo s
```
![](https://bingxiangstorage.blob.core.windows.net/blog/hexo-init/2022-02-01-20-20-07.png)

    確認部落格是否建立成功 http://localhost:4000/

    修改 部落格樣式 (以 3-hexo 為例)

    1. Clone SourceCode

      git clone https://github.com/yelog/hexo-theme-3-hexo.git themes/3-hexo

    2. 修改 /_config.yml

      3-hexo 自帶highlight 故關掉預設highlight

      - highlight:
-   enable: true
+ highlight:
+   enable: false

      修改 theme

      - theme: landscape
+ theme: 3-hexo

      確認是否成功

    上傳至 GitHub Page

    1. 建立Github 儲存庫 https://github.com/new
      儲存庫名稱為 {githubAccountName}/.github.io
      設為Public

    githubAccountName 請用個人github帳戶代替

    1. 安裝 hexo-deployer-git
      hexo deploy時 自動推送靜態檔案至GitHub儲存庫

      npm install hexo-deployer-git --save

    2. 修改 /_config.yml

    - deploy:
-  type: ''

+ deploy:
+   type: git
+   repo: https://github.com/{githubAccountName}}/{githubAccountName}.github.io.git
+   branch: master
    1. 佈署
      hexo deploy
