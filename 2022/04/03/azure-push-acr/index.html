    上傳Docker Image至 Azure Container Registry

    Devops > Azure Azure
    發布時間 : 2022-04-03 22:51
    字數:123 觀看數 :
    評論:
    1. 紀錄一下 上傳Docker Image至 Azure Container Registry

    紀錄一下 上傳Docker Image至 Azure Container Registry

    #  1. Azure 登入
azure login

#  2. 建立ACR
#  n = name(ACR名稱), g = resurece-group(azure帳號的資源群組), l = 地區(location), sku = 等級
az acr create -n bxacrtest -g BX -l southeastasia --sku basic

#  3. 查看ACR資訊列表
Az acr list -o table

#  4. 登入ACR
az acr login -n BX

#  5. 上傳前要設定Docker Image tag的前綴
docker tag k8ssample  bxacrtest.azurecr.io/k8ssample

#  6. 上傳Docker Image至ACR
docker push bxacrtest.azurecr.io/k8ssample

#  7. 查看ACR裡的Docker Image
az acr repository list -n bxacrtest
